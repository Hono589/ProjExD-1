# 第3回
## 迷路ゲーム : 迷えるこうかとん(ex03/maze.py)
### ゲームの概要
 - ex03/maze.pyを実行すると、1500x900に迷路が描写され、迷路に沿ってこう 
   かとんを移動させるゲーム
 - 実行するたびに迷路の構造は変化する

### 操作方法
  - 矢印キーでこうかとんを上下左右に移動する
### 追加機能
 - 敵をランダムで出力 
 - こうかとんを別のキャラに変更
